# 第 8 章 章后习题及答案

> 来源依据：教材第 8 章习题；第 8 章 PPT；`知识点汇总.txt` 第八章。

## 8.1 名词解释要点

本章名词可按四组记忆：总线类型、信号组成、性能指标、传输控制。

- **系统总线/I/O 总线/外部总线**：不同层次的互连通道。
- **地址总线/数据总线/控制总线**：分别传地址、数据和控制信息。
- **主设备/从设备/总线主控**：发起事务者、响应者和控制总线者。
- **总线宽度/总线时钟/总线周期/总线带宽**：计算性能的关键指标。
- **DDR/QDR**：每时钟传输多次数据。
- **突发传输**：一次地址后连续传多个数据。
- **同步/异步定时**：靠统一时钟或握手信号协调。

## 8.2 简答题答案要点

1. 总线结构减少连线数量、降低成本、方便扩展，但共享通道可能成为瓶颈。
2. 单总线简单但冲突多；双总线分担流量；三总线进一步提高并行性和扩展性。
3. 传输方式有串行、并行、并串行等；串行线少抗干扰强，并行一次传多位但高速受偏斜限制。
4. 集中式仲裁有链式查询、计数器定时查询、独立请求；分别在硬件成本、优先级灵活性和响应速度上取舍。
5. 突发传输给一次首地址后连续传多个数据，优点是减少地址阶段开销。
6. 影响性能的因素有总线宽度、频率、传输次数、等待周期、仲裁开销、传输模式。
7. 总线结构成本低、扩展方便，但共享带宽；交换结构并行性强但成本和控制复杂度更高。

## 8.4 总线最大速率

100MHz、32 位、每周期传 1 个数据：

```text
32 位 = 4B
最大速率 = 4B × 100MHz = 400MB/s
```

提高一倍可采用：总线宽度加倍到 64 位、频率加倍到 200MHz、采用双沿传输，或通过突发传输减少非数据周期开销。

## 8.5 异步通信速率

若每字符包括 1 位起始位、8 位数据位、1 位校验位、1 位停止位，共 11 位：

```text
字符速率 = 4800 / 11 ≈ 436 字符/秒
每位时间 = 1 / 4800 ≈ 208.3 μs
有效数据位速率 = 436 × 8 ≈ 3488 bit/s
```

## 8.6 至 8.7 仲裁电路题

- 链式查询按优先级把设备串接，响应优先级最高的设备最靠近仲裁器。
- 独立请求方式中，每个设备有独立请求线，仲裁器内部用优先级逻辑决定授权线。

画图时重点标出请求线、授权线、优先级顺序和设备连接关系。

## 8.8 地址/数据复用总线

16 位地址/数据复用，地址空间为：

```text
2^16 个可寻址单元
```

若一次事务传 16 位数据，需要 4 个时钟周期，时钟 8MHz：

```text
事务时间 = 4 / 8MHz = 0.5 μs
数据量 = 2B
传输率 = 2B / 0.5μs = 4MB/s
```

## 8.9 突发读写速率

时钟 50MHz，周期 20ns，32 位总线每数据阶段传 4B，8 个数据阶段共 32B。

读事务：

```text
1 地址 + 3 等待 + 8 数据 = 12 周期 = 240ns
速率 = 32B / 240ns ≈ 133.3MB/s
```

写事务：

```text
1 地址 + 2 等待 + 8 数据 + 2 恢复 = 13 周期 = 260ns
速率 = 32B / 260ns ≈ 123.1MB/s
```

## 8.10 至 8.11 数据块传输题

通用步骤：

1. 先算一次事务传输的数据量。
2. 再算地址阶段、数据阶段、等待或恢复阶段总周期。
3. 若存储器访问可与总线传输重叠，总时间取二者中较长的关键路径。
4. 总线带宽用总有效数据量除以总时间。
5. 每秒事务次数用事务次数除以总时间。
