# 原码与补码一位乘法

原码一位乘法相当于竖式计算，考试不考。下面介绍补码一位乘法的 Booth 算法。

本题的完整答案存于`第二次作业.pdf`。 

## 一句话理解

Booth 算法的核心思想是：不需要提前单独处理符号位，而是将符号位和数值位一起参与运算。通过观察乘数 $y$ 的最后两位状态，决定是对部分积**加 $x$**、**减 $x$** 还是**加 $0$**，然后统一进行算术右移，经过 $n$ 次循环得出最终结果。

## 一、 题目解析与前期准备

**已知条件：**$x = 0.10110$、$y = -0.00011$，求 $x \cdot y$ 的补码结果。

**第一步：求出相关的补码（双符号位）**

为了在相加时防止溢出，部分积和被乘数 $x$ 通常采用**双符号位**，乘数 $y$ 采用**单符号位**即可。

- $[x]_补 = 00.10110$

- $[-x]_补 = 11.01010$

- $[y]_补 = 1.11101$

## 二、 核心运算规则 (Booth 规则)

算法需要设置一个**附加位**，记为 $y_{n+1}$，初始值为 $0$。

每次运算，我们从右向左观察乘数 $[y]_补$ 的当前位 $y_n$ 和它的附加位 $y_{n+1}$（即判断 $y_n y_{n+1}$ 的组合）：

| **判断依据 ($y_n y_{n+1}$​)** | **执行操作**                         |
| ------------------------- | -------------------------------- |
| **`10`**                  | 部分积 $\Sigma$ **加上 $[-x]_补$**     |
| **`01`**                  | 部分积 $\Sigma$ **加上 $[x]_补$**      |
| **`00` 或 `11`**           | 部分积 $\Sigma$ **加 $0$** (即只移位不加减) |

> **移位规则：** 每次执行完加法后，部分积 $\Sigma$ 和乘数寄存器必须整体**算术右移 1 位。此时会空出一位，为了保持负数依然是负数，正数依然是正数，必须拿符号位去填充空缺。

## 三、 逐步推导过程解析

我们在脑海中划定两个区域：左边是**部分积 $\Sigma$**（初始为 $00.00000$），右边是**乘数及其附加位**。图片中用不同颜色标记了每次移位后被“挤”出来的数字。

- **初始状态：**
  
  $\Sigma = 00.00000$
  
  乘数及附加位 = $1.11101\mathbf{0}$ （最后两位是 **`10`**）

### 第 1 步：判定 `10`

- **操作：** 加 $[-x]_补$
  
  $00.00000 + 11.01010 = 11.01010$

- **移位：** 算术右移 1 位
  
  $\Sigma$ 变为 $11.10101$，最低位的 `0` 被移出。
  
  图片记为：`-> 11.10101` (移出位: `0`)

- **下一个判定目标：** 乘数剩余 `1.1110`，结合刚移出的 `1`，最后两位变为 **`01`**。

### 第 2 步：判定 `01`

- **操作：** 加 $[x]_补$
  
  $11.10101 + 00.10110 = 00.01011$

- **移位：** 算术右移 1 位
  
  $\Sigma$ 变为 $00.00101$，最低位的 `1` 被移出。
  
  图片记为：`-> 00.00101` (移出位累积: `10`)

- **下一个判定目标：** 乘数剩余 `1.111`，结合移位，最后两位变为 **`10`**。

### 第 3 步：判定 `10`

- **操作：** 加 $[-x]_补$
  
  $$00.00101 + 11.01010 = 11.01111$$

- **移位：** 算术右移 1 位
  
  $\Sigma$ 变为 $11.10111$，最低位的 `1` 被移出。
  
  图片记为：`-> 11.10111` (移出位累积: `110`)

- **下一个判定目标：** 最后两位变为 **`11`**。

### 第 4 步：判定 `11`

- **操作：** 加 $0$（图片中省略了加法步骤，直接进行移位）

- **移位：** 对上一步的结果直接算术右移 1 位
  
  $\Sigma$ 变为 $11.11011$，最低位的 `1` 被移出。
  
  图片记为：`-> 11.11011` (移出位累积: `1110`)

- **下一个判定目标：** 最后两位变为 **`11`**。

### 第 5 步：判定 `11`

- **操作：** 加 $0$

- **移位：** 再次直接算术右移 1 位
  
  $\Sigma$ 变为 $11.11101$，最低位的 `1` 被移出。
  
  图片记为：`-> 11.11101` (移出位累积: `11110`)

## 四、 最终结果合并

由于乘数 $y$ 有 5 位数值位，按照 Booth 算法，移位需要进行 5 次。计算到此结束。

我们将最后的部分积 $\Sigma$ 和所有被移出的低位拼接在一起（图中的绿色框）：

- 部分积：$11.11101$

- 移出位：$11110$

**最终结果：** $[x \cdot y]_补 = 11.1110111110$
