# 原码与补码一位乘法

> 来源依据：`知识点汇总.txt` 第三章；第 3 章 PPT“原码一位乘法”“补码一位乘法”；教材第 3 章。

## 一句话理解

一位乘法就是把乘法拆成很多次“看乘数最低位、决定是否加被乘数、再移位”的过程。

## 基本概念

先想普通十进制竖式乘法：乘数某一位是几，就产生对应的一行部分积。二进制更简单，因为乘数每一位只可能是 `0` 或 `1`：

```text
乘数位是 0：这一位不贡献部分积
乘数位是 1：这一位贡献一个被乘数
```

所以二进制乘法可以用加法和移位完成。硬件每次只处理乘数的一位，这就是“一位乘法”。

两类方法要分清：

| 方法 | 符号怎样处理 | 数值部分怎样做 |
| --- | --- | --- |
| 原码一位乘法 | 符号位单独处理 | 对绝对值做无符号乘法 |
| 补码一位乘法 | 符号参与补码运算 | 常用 Booth 思想，根据相邻位决定加、减或不动 |

## 工作原理或推导过程

### 1. 原码一位乘法怎么想

原码把符号位和数值位分得很清楚，所以乘法也常分两步：

1. 结果符号位由两个操作数符号位异或得到。
2. 数值部分只看绝对值，用二进制无符号乘法完成。

符号位异或的意思是：

```text
同号相乘为正，异号相乘为负
```

数值部分的循环步骤：

1. 部分积初值为 0。
2. 检查乘数最低位。
3. 若最低位为 1，部分积加被乘数；若为 0，不加。
4. 部分积和乘数整体右移。
5. 重复 n 次，n 是乘数数值位位数。

为什么右移？因为每处理完乘数的一位，就相当于把下一位移到“当前最低位”位置，让硬件继续用同一套加法器和寄存器处理。

### 2. 补码 Booth 一位乘法怎么想

补码乘法可以直接处理负数。Booth 方法的直觉是：不要只看当前位是不是 `1`，而是看乘数位串从 `0` 到 `1`、从 `1` 到 `0` 的变化。

一串连续的 `1` 可以改写成一次加和一次减。例如：

```text
00111100B = 01000000B - 00000100B
```

这说明连续多个 `1` 不一定要每一位都加一次被乘数，可以在边界处处理。Booth 法就是利用这个思想。

常用判断表如下，设当前最低位为 `Q0`，附加位为 `Q-1`：

| `Q0 Q-1` | 操作 | 直观含义 |
| --- | --- | --- |
| `00` | 不操作 | 仍在 0 段中 |
| `11` | 不操作 | 仍在 1 段中 |
| `01` | 加被乘数 | 从 1 段结束处补偿 |
| `10` | 减被乘数 | 从 0 到 1 的开始处处理 |

Booth 法常见步骤：

1. 部分积寄存器 `A` 置 0。
2. 乘数放在 `Q` 中，附加位 `Q-1` 置 0。
3. 根据 `Q0Q-1` 决定 `A` 加被乘数、减被乘数或不变。
4. `A、Q、Q-1` 整体算术右移一位。
5. 重复 n 次。

算术右移要保持符号位，因为补码乘法的中间结果可能为负。

## 典型例子

### 例 1：用原码思想理解 `5 × 3`

写成二进制：

```text
5 = 0101B
3 = 0011B
```

乘数 `0011` 从低位看是两个 `1`：

```text
第 0 位是 1：加 0101
第 1 位是 1：再加左移 1 位的 0101
```

所以：

```text
0101 + 1010 = 1111B = 15
```

硬件的一位乘法只是把这个竖式过程改成“每轮加一次或不加，然后移位”。

### 例 2：原码乘法的符号处理

若计算：

```text
(-5) × (+3)
```

符号位：

```text
1 ⊕ 0 = 1
```

所以结果为负。数值部分仍然算：

```text
5 × 3 = 15
```

最终结果是：

```text
-15
```

### 例 3：Booth 表怎么看

如果某轮看到：

```text
Q0Q-1 = 10
```

说明乘数位串出现了从 `0` 到 `1` 的边界，按表要执行：

```text
A = A - M
```

如果下一轮看到：

```text
Q0Q-1 = 11
```

说明仍在连续的 `1` 段中，不需要重复加减，只做算术右移。

## 常见考法

- 用原码一位乘法列出每一轮部分积、乘数和移位结果。
- 给两个带符号数，用补码 Booth 一位乘法求积。
- 判断原码乘法中符号位是否应参与数值运算。
- 判断 Booth 法中 `00、01、10、11` 分别对应什么操作。
- 判断乘法结果是否超出给定位数。

## 易错点

- 原码乘法的符号位通常单独处理，不参加数值部分累加。
- 原码数值部分按绝对值算，最后再加符号。
- Booth 法看的是 `Q0Q-1` 两位，不是只看 `Q0`。
- 补码乘法右移要用算术右移，保持符号。
- 表格题每轮都要写清“加/减/不操作”和“右移”，漏一步结果就会错。

## 小结

- 二进制乘法的本质是部分积相加，乘数位为 1 才贡献部分积。
- 原码一位乘法适合“先定符号，再算绝对值”。
- 补码 Booth 乘法利用相邻位变化，直接处理带符号数。
- 乘法器的基础仍然是加法器和移位器。
