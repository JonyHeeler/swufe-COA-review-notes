## 1.1 名词解释要点

- **摩尔定律**：集成电路上晶体管数量随工艺进步呈周期性增长的经验规律。
- **汇编器**：把汇编语言程序翻译成机器语言目标代码的软件。
- **编译器**：把高级语言源程序翻译成目标代码的软件。
- **解释器**：边解释边执行高级语言程序的软件。
- **链接器**：把多个目标文件和库文件连接成可执行文件的软件。
- **时钟周期**：CPU 时钟信号的一个周期，是处理器同步工作的基本时间单位。
- **机器字长**：CPU 一次能够处理的二进制数据位数，常与寄存器宽度相关。
- **主存容量**：主存储器能够保存的二进制信息总量。
- **CPI**：每条指令平均消耗的时钟周期数。
- **IPC**：每个时钟周期平均完成的指令条数。
- **MIPS**：每秒执行百万条指令数。
- **MFLOPS**：每秒执行百万次浮点运算数。
- **CPU 时间**：程序实际占用 CPU 执行的时间。

## 1.3 冯诺依曼结构

基本思想：程序和数据都用二进制表示并存放在存储器中，计算机按地址取指令，译码后自动执行。

硬件系统包括运算器、控制器、存储器、输入设备和输出设备。运算器完成算术逻辑运算，控制器产生控制信号，存储器保存程序和数据，输入输出设备负责外部交互。

## 1.4 计算机系统层次

可分为硬件层、机器语言层、操作系统层、语言处理层和应用层。硬件层提供物理执行能力，机器语言层直接面向指令，操作系统管理资源，高级语言和应用层让用户用更自然的方式表达任务。

## 1.5 MIPS 计算

平均 CPI：

```text
CPI1 = 0.4×2 + 0.2×3 + 0.15×4 + 0.25×5 = 3.25
CPI2 = 0.4×2 + 0.2×2 + 0.15×3 + 0.25×4 = 2.65
```

MIPS：

```text
MIPS1 = 600 / 3.25 ≈ 184.62
MIPS2 = 800 / 2.65 ≈ 301.89
```

计算机 2 的 MIPS 更高。

## 1.6 编译优化前后

设优化前总指令数为 `N`。优化前平均 CPI：

```text
0.4×1 + 0.2×2 + 0.15×2 + 0.25×2 = 1.6
```

优化后 A 类指令减半，总指令数变为 `0.8N`，总周期数系数为：

```text
0.2×1 + 0.2×2 + 0.15×2 + 0.25×2 = 0.9
```

优化后平均 CPI：

```text
0.9 / 0.8 = 1.125
```

MIPS：

```text
优化前 MIPS = 500 / 1.6 = 312.5
优化后 MIPS = 500 / 1.125 ≈ 444.44
```

结论：优化后平均 CPI 和 MIPS 都改善，但真正衡量程序是否变快，应看 CPU 时间。优化后 CPU 时间为原来的 `0.9/1.6=0.5625`，速度提升更明显。
