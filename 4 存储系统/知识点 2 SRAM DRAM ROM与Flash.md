# SRAM、DRAM、ROM 与 Flash

> 来源依据：第 4 章 PPT“半导体存储器”“SRAM”“DRAM”“只读存储器”“闪速存储器”。

## 一句话理解

SRAM、DRAM、ROM 和 Flash 的区别，本质是在“速度、容量、成本、掉电是否丢失、能否方便改写”之间取舍。

## 基本概念

先抓住两个最重要的分类词：

- **易失性**：断电后数据会丢失。
- **非易失性**：断电后数据仍能保存。

再看常见半导体存储器：

| 类型 | 是否易失 | 主要特点 | 常见用途 |
| --- | --- | --- | --- |
| SRAM | 易失 | 快、不需刷新、成本高、容量较小 | Cache |
| DRAM | 易失 | 容量大、成本低、需要刷新 | 主存 |
| ROM | 非易失 | 主要用于保存固定程序或固件 | BIOS/固件 |
| Flash | 非易失 | 可电擦写，读快写慢，常按块擦除 | U 盘、SSD |

一个直观理解：

- SRAM 像“稳定的小开关”，状态保持得住，但硬件多，所以贵。
- DRAM 像“会慢慢漏水的小水杯”，便宜又能做很多个，但要定期补水，也就是刷新。
- ROM 像“出厂写好的说明书”，通常用来保存不常改变的内容。
- Flash 像“可擦写的电子笔记本”，断电不丢，但擦写没有 RAM 那么随意。

## 工作原理或推导过程

### 1. SRAM 为什么快但贵

SRAM 通常用触发器保存 1 位数据。触发器只要不断电，就能稳定保持 0 或 1。

因此 SRAM：

- 读写速度快。
- 不需要周期性刷新。
- 每一位需要较多晶体管，面积大、成本高。
- 容量不适合做得特别大。

这就是为什么 Cache 常用 SRAM：Cache 容量不大，但要求极快。

### 2. DRAM 为什么要刷新

DRAM 通常用“电容是否带电”表示 1 位数据。

问题是电容会漏电。如果不管它，原来表示 1 的电荷会慢慢消失，数据就错了。

所以 DRAM 必须定期刷新：

```text
读出一行数据 -> 再写回去 -> 恢复电荷
```

DRAM 的特点：

- 每位结构简单，适合做大容量。
- 成本低，适合作主存。
- 速度比 SRAM 慢。
- 需要刷新，刷新期间会占用一定存储器时间。

### 3. ROM 的“只读”怎么理解

ROM 的重点不是“永远不能改”，而是**正常运行时主要用于读取，写入或改写不如 RAM 灵活**。

常见 ROM 相关类型：

| 类型 | 直观理解 |
| --- | --- |
| 掩膜 ROM | 制造时写好，之后不能普通改写 |
| PROM | 用户可写一次 |
| EPROM | 可擦除后重写，通常需特殊条件 |
| EEPROM | 可电擦除重写 |

考试一般不要求展开太细，但要知道 ROM 通常是非易失的。

### 4. Flash 为什么常用于 U 盘和 SSD

Flash 属于非易失性存储器，断电后数据仍能保留，并且可以电擦写。

但 Flash 和 RAM 不同：

- 读取较快。
- 写入比读取慢。
- 擦除通常按块进行，不是像 RAM 那样随便改某一位。
- 有擦写寿命限制。

所以 Flash 适合做外存或固态存储，不适合直接替代主存中的 DRAM。

## 典型例子

### 例 1：为什么 Cache 用 SRAM，主存用 DRAM

Cache 离 CPU 最近，容量相对小，但速度要求极高，所以用 SRAM。

主存容量需要很大，如果全部用 SRAM，成本和面积都太高，所以用 DRAM。

这体现了存储层次的折中：

```text
SRAM：快，但贵
DRAM：慢一些，但便宜且容量大
```

### 例 2：判断是否需要刷新

题目问：SRAM、DRAM、ROM、Flash 中哪些需要刷新？

答案通常是：

```text
DRAM 需要刷新
SRAM 不需要刷新
ROM 和 Flash 也不按 DRAM 方式刷新
```

因为刷新是为了解决 DRAM 电容漏电问题。

### 例 3：RAM 是否等于随机访问且掉电丢失

RAM 的本意是随机访问存储器，表示可以按地址直接访问，而不是只能顺序读。

常见 RAM 如 SRAM、DRAM 通常是易失性的，但“随机访问”和“掉电丢失”是两个不同角度的概念，不要把定义混成一个。

## 常见考法

- 比较 SRAM 和 DRAM 的速度、容量、成本和刷新需求。
- 判断哪些存储器掉电后数据会丢失。
- 判断哪些存储器适合做 Cache、主存、固件或 SSD。
- 判断 ROM 是否可以随机读取。
- 解释 DRAM 为什么要刷新。
- 结合存储器扩展题计算 DRAM 刷新周期或刷新间隔。

## 易错点

- **SRAM 不需要刷新，但断电仍会丢失数据**。
- **DRAM 需要刷新，不是因为 CPU 经常访问它，而是因为电容会漏电**。
- **ROM 的“只读”不等于只能顺序读**，ROM 也可以按地址随机读取。
- **Flash 可擦写，但不能像普通 RAM 那样高速、任意、无限次写入**。
- **主存常用 DRAM，不是因为 DRAM 比 SRAM 快，而是因为容量和成本更合适**。

## 小结

- SRAM：快、贵、不刷新、常作 Cache。
- DRAM：容量大、便宜、要刷新、常作主存。
- ROM：非易失，常保存固件。
- Flash：非易失、可擦写，常用于 U 盘、SSD。
- 判断存储器类型时，一定分清速度、容量、成本、易失性和刷新需求。
