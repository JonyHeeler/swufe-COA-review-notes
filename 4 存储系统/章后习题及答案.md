# 第 4 章 章后习题及答案

> 来源依据：教材第 4 章习题；第 4 章 PPT；`知识点汇总.txt` 第四章。

## 4.1 名词解释要点

本章名词建议分为三组：主存与芯片、cache、虚拟存储器。

- **存取时间/存取周期/存储器带宽**：分别描述单次访问延迟、两次访问最小间隔、单位时间传输量。
- **大端/小端**：多字节数据在连续地址中的高低字节存放顺序。
- **SRAM/DRAM/刷新**：SRAM 不需刷新，DRAM 需周期性刷新。
- **位扩展/字扩展**：分别扩展字长和字数。
- **直接相联/全相联/组相联**：cache 三种主存块放置规则。
- **TLB/页表/页框号**：虚实地址转换相关结构。
- **LRU/LFU/写回/写穿**：替换与写策略。

## 4.3 简答题答案要点

1. 层次化存储体系用于在速度、容量和成本之间折中。
2. 片选端用于确定当前访问由哪个芯片响应。
3. DRAM 电容电荷会泄漏，需要按行周期刷新，可集中、分散或异步刷新。
4. 多体交叉把连续地址分散到不同体，使多个体交错工作。
5. cache 的存在和替换对程序员不可见，所以说透明。
6. 直接相联每个主存块位置唯一，不存在多个候选行，因此不需要替换算法。
7. 多级 cache、主存和 I/O 可能保存同一数据副本，需要一致性策略。
8. FIFO 简单但可能淘汰热点块；LRU 效果好但实现复杂；LFU 看访问次数但对阶段性变化不敏感；随机算法简单但命中率不稳定。

## 4.4 地址寄存器与数据寄存器

32KB 按 16 位字编址：

```text
32KB = 32768B
16 位字 = 2B
字数 = 32768/2 = 16384 = 2^14
```

地址寄存器 14 位，数据寄存器 16 位。

## 4.5 至 4.7 存储器扩展

通用做法：

```text
并联片数 = 目标字长 / 芯片字长
分组数 = 目标字数 / 芯片字数
总片数 = 并联片数 × 分组数
```

画连接图时，低位地址线连芯片内部地址端，高位地址线经译码器产生片选信号，数据线按位宽并联。

## 4.8 至 4.10 DRAM 刷新

刷新题先确定行数。若每行刷新间隔不超过 `T`，有 `R` 行，则异步刷新信号间隔为：

```text
T / R
```

集中刷新时，刷新一遍至少需要 `R` 个读写周期，CPU 死时间为：

```text
R × 读写周期
```

## 4.11 至 4.14 cache 综合题

解题顺序：

1. 求主存块号和块内偏移。
2. 根据映射方式求 cache 行号或组号。
3. 写出 tag。
4. 按访问序列模拟命中、缺失和替换。
5. 命中率为 `命中次数 / 总访问次数`。

数组局部性题中，行优先存放时按行扫描空间局部性好，按列扫描空间局部性差；循环变量和 `sum` 往往有时间局部性。

## 4.15 至 4.19 虚拟存储器综合题

虚实地址转换模板：

```text
页大小 = 2^pB
VPO/PPO = p 位
VPN = 虚拟地址位数 - p
PPN = 物理地址位数 - p
```

访问流程：

```text
虚拟地址 → TLB → 页表 → 物理地址 → cache → 主存
```

若页表装入位为 0，则缺页；若 TLB 命中，则页一定在主存；若 cache 缺失，只需从主存取块，开销远小于从外存调页。
