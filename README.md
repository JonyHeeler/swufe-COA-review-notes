# 西南财经大学《计算机组成原理》课程复习资料

本仓库整理西南财经大学《计算机组成原理》课程复习资料，按 8 个章节组织。资料优先覆盖 `知识点汇总.txt` 中列出的核心考点，再补充教师 PPT 中出现但核心清单未覆盖的知识点。

本套资料只保证适用于 25-26-2 学期的《计算机组成原理》期末考试，时间为 2026 年 6 月 30 日，在此之后本仓库将停止维护。

## 目录结构

| 章节    | 文件夹          | 核心复习方向                        |
| ----- | ------------ | ----------------------------- |
| 第 1 章 | `1 计算机系统概述`  | CPI、MIPS、CPU 时间、计算机系统层次       |
| 第 2 章 | `2 数据信息的表示`  | 机器数、进制转换、IEEE754、校验码、C 语言数据表示 |
| 第 3 章 | `3 运算方法与运算器` | 补码运算、乘除法、加法器、浮点运算             |
| 第 4 章 | `4 存储系统`     | 主存编址、Cache、虚拟存储器              |
| 第 5 章 | `5 指令系统`     | 扩展指令码、寻址方式、MIPS 指令            |
| 第 6 章 | `6 中央处理器`    | 数据通路、控制器、指令执行时间               |
| 第 7 章 | `7 指令流水线`    | 五段流水线、冲突、冒险、性能计算              |
| 第 8 章 | `8 总线系统`     | 总线带宽、传输方式、仲裁、总线结构             |

每章通常包含：

- 教师 PPT：如 `1 计算机系统概述.pptx`，用于回看课堂原始内容。
- `本章复习导读.md`：先帮你判断本章哪些内容最重要、应该按什么顺序复习。
- `核心知识点 x 知识点名字.md`：来自核心知识点清单，是每章最优先掌握的内容。
- `知识点 x 知识点名字.md`：来自 PPT 的补充知识点，用来补齐背景概念和课堂覆盖内容。
- `习题 x.md`：课后习题的答案。
- 三次作业的解析讲义已放在对应章节。

## 推荐使用顺序

1. 先读对应章节的 `本章复习导读.md`，明确这一章到底考什么、先学什么、后学什么。
2. 再读 `核心知识点 x ...md`。这些文件是复习主线，建议优先掌握里面的定义、公式、推导过程、典型例题和易错点。
3. 然后读 `知识点 x ...md` 和补充专题文件。它们主要用于补齐 PPT 中出现但核心清单没有单独列出的内容，例如编码、硬件结构、移位、加法器等背景知识。
4. 最后做对应章节的 `习题 x.md`。做题时不要只看答案，要重点回到前面的知识点文件，确认每一步计算为什么这样做。
5. 如果本章有单独作业材料，先看对应知识点中的例题讲解，再把作业文件当成强化练习。第 2 章的第一次作业已有单独讲解；第 3 章的第二次作业主要和乘除法、补码运算等知识点配合使用。
6. 需要查原始材料时，再回看根目录的 `原版教材.pdf`、`课后习题.pdf` 或章节 PPT。
